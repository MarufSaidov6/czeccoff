# Чехов — RSVP-читалка для быстрого чтения

Читалка для быстрого чтения методом **RSVP** (Rapid Serial Visual Presentation): слова показываются по одному в фиксированной точке экрана, якорная буква (ORP) подсвечена красным и всегда стоит в одной позиции. Глаза не совершают саккад — скорость чтения растёт с ~250 до 400–700 слов/мин.

В отличие от аналогов (Spritz, Reedy), читалка борется с главной слабостью RSVP — потерей понимания на скорости: умный темп, перемотка по предложениям, карта регрессий и адаптивный тренер скорости.

## Возможности

- **Пиксельный ORP-алгоритм** — якорь считается по реальной ширине букв в текущем шрифте, как у Spritz (а не упрощённый индексный). Три профиля позиции якоря под кириллицу.
- **Умный темп** — паузы на конце предложения, запятых, границах абзаца, числах и длинных словах; плавный разгон после паузы.
- **Три режима смены слов** — резкая, микро-промежуток (ISI), плавная — для честного сравнения.
- **Навигация** — перемотка по предложениям, прыжок по клику на прогресс-бар, контекст текущего предложения на паузе.
- **Тренер скорости + карта регрессий** — темп растёт за чистое чтение, запоминаются места, к которым вы возвращались.
- **Реальные файлы** — вставка текста, `.txt`, `.fb2` (автодетект кодировки, включая windows-1251; «грязные» файлы с lib.ru).
- **Прогресс сохраняется** локально в браузере (IndexedDB + localStorage). Без сервера, без аккаунтов, без сборки.

## Как запустить

**Онлайн (для всех):** откройте ссылку демо выше — приложение работает прямо в браузере.

**Локально:** скачайте репозиторий и откройте `index.html` двойным кликом. Всё в одном файле, зависимостей нет.

## Книги

В папке [`sample-books/`](sample-books/) лежит **стартовый набор** — пара коротких демо-текстов и рассказы-классики в общественном достоянии, чтобы сразу попробовать читалку. На стартовом экране они показываются кнопками «Попробуй на примере» (работают, когда приложение открыто по ссылке / через GitHub Pages).

**Добавить свою книгу в набор:**
1. Положите `.txt`-файл в `sample-books/`.
2. Допишите его в `sample-books/manifest.json`:
   ```json
   { "file": "moya-kniga.txt", "title": "Моя книга", "author": "Автор", "note": "описание" }
   ```
3. Закоммитьте и запушьте — кнопка появится автоматически.

**Свои личные книги, которые не нужно публиковать,** не кладите в открытый репозиторий: просто открывайте их кнопкой «Открыть файл (.fb2 / .txt)» — они останутся только в вашем браузере. Папка `my-books/` добавлена в `.gitignore` на этот случай.

Михаловского), взятые с [ru.wikisource.org](https://ru.wikisource.org) и находящиеся в общественном достоянии (лицензии PD проверены). Если добавляете чужие книги в **публичный** репозиторий, выкладывать можно только то, что в общественном достоянии (public domain — обычно авторы, умершие более 70 лет назад: Чехов, Пушкин, Толстой и т.п.) или распространяется по свободной лицензии. Защищённые копирайтом книги в открытый репозиторий выкладывать нельзя — держите их приватно (см. выше).

## Технологии

Один файл `index.html` — ванильный HTML/CSS/JS, без зависимостей и без сборки. Canvas-метрики для ORP, DOMParser для fb2, IndexedDB для хранения текстов.

## Тесты

```bash
node tests/run.js
```

Покрывают токенизацию, определение границ предложений, ORP-алгоритм, умные паузы, краевую пунктуацию, смену слов, парсинг fb2-кодировок и подсчёт очков турнира (47 тестов).

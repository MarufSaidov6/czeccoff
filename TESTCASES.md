# Flash Reader — Test Cases

**Версия:** 1.0 · соответствует PRD 1.0 · **Дата:** 2026-06-10
**Объект тестирования:** `index.html` (одностраничное приложение, без сборки)
**Приоритеты:** P0 — ядро гипотезы, блокер релиза · P1 — важная функциональность · P2 — полировка
